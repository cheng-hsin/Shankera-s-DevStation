## 前言

Windows 2012 R2 是一個很老的 Windows 版本，但是在某些情況下，還是會需要使用到，例如在某些公司的環境中，還是會使用到這個版本的 Windows。
AWS 提供了兩種方式可以連線到 Windows 2012 R2，分別是 ssh 和 rdp，這篇文章會介紹這兩種方式。
