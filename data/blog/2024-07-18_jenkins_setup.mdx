# Install Jenkins

To install Jenkins, you can follow the steps below:

Step 1: Update your system

```bash
sudo apt-get update
```

Step 2: Install Java

```bash
sudo apt-get install openjdk-8-jdk
```

Step 3: Add the Jenkins repository key to the system

```bash
wget -q -O - https://pkg.jenkins.io/debian/jenkins.io.key | sudo apt-key add -
```

Step 4: Add the Jenkins repository to the system

```bash
sudo sh -c 'echo deb http://pkg.jenkins.io/debian-stable binary/ > /etc/apt/sources.list.d/jenkins.list'
```

Step 5: Install Jenkins

```bash
sudo apt-get update
sudo apt-get install jenkins
```

Step 6: Start Jenkins

```bash
sudo systemctl start jenkins
```

Step 7: Check the status of Jenkins

```bash
sudo systemctl status jenkins
```

Step 8: Open Jenkins in your browser

You can open Jenkins in your browser by visiting `http://localhost:8080`.

# Other Jenkins commands

## Restart Jenkins

```bash
sudo systemctl restart jenkins
```

## Stop Jenkins

```bash
sudo systemctl stop jenkins
```

## Check the logs of Jenkins

```bash
sudo journalctl -xe
```

## Unlock Jenkins

When you first install Jenkins, you need to unlock it by running the following command:

```bash
sudo cat /var/lib/jenkins/secrets/initialAdminPassword
```

This will show you the initial password to unlock Jenkins.

## Install Jenkins plugins

You can install Jenkins plugins by following these steps:

1. Open Jenkins in your browser
2. Click on `Manage Jenkins`
3. Click on `Manage Plugins`
4. Click on the `Available` tab
5. Search for the plugin you want to install
6. Click on `Install without restart`

## Update Jenkins

You can update Jenkins by running the following command:

```bash
sudo apt-get update
sudo apt-get install jenkins
```

This will update Jenkins to the latest version.

## Backup Jenkins

You can backup Jenkins by copying the Jenkins home directory to a safe location. The Jenkins home directory is usually located at `/var/lib/jenkins`.

## Restore Jenkins

You can restore Jenkins by copying the Jenkins home directory back to the original location.

## Uninstall Jenkins

You can uninstall Jenkins by running the following command:

```bash
sudo apt-get remove jenkins
```

This will remove Jenkins from your system.
